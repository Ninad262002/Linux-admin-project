# 🐧 User Info Checker Script

A simple Linux shell script to display user and system information.

## 📌 Features
- Displays current logged-in user
- Shows user ID (UID, GID)
- Prints system information
- Displays current working directory

## 🛠️ Technologies Used
- Bash (Shell Scripting)
- Linux Commands (whoami, id, uname, pwd)

## ▶️ How to Run

### Step 1: Clone the repository
git clone https://github.com/your-username/Linux-admin-project.git

### Step 2: Navigate to project
cd Linux-admin-project/user-info-checker

### Step 3: Give permission
chmod +x userinfo.sh

### Step 4: Run script
./userinfo.sh
